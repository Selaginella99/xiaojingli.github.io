# 简约版系统的优势

- 简约
- 安静
- 稳定
- 搞笑
- 贴心

# 最佳应用软件清单

- Termial 的替代： iTerm
- 编辑器：TextMate & Sublime Text
- 存储类云盘：Dropbox
  - 可以在开学季参加Campus活动，可以免费获得25G的容量
- 存放私人「卡片」：Evernote
 - 支持较多的插件，使用更为方便
- 搜索工具：Spotlight + Alfred/Everything
  
## IFTTT 推送到 Kindle  
   1. 对 Dopbox, Gmail 进行授权
   2. 确认连接
   3. 登陆 amazon.cn 将gmail添加至「已认可的发件人电子邮箱列表」
   4. 在 IFTTT 中的「 send to 」填入 kindle 的邮箱地址
